Why GQLS

Built for the way you actually work.

⚡

curl import

Paste a raw curl command from DevTools. Headers, auth tokens, and body are extracted automatically.

📄

SARIF output

Integrates with GitHub Code Scanning, GitLab SAST, and any CI pipeline that speaks SARIF 2.1.0.

🎯

12 targeted checks

GraphQL-specific detections from introspection abuse to unauthenticated mutations and SQL injection.

🔧

CI-ready

Exit codes, --fail-on severity threshold, and JSON/SARIF output built for automated pipelines.

🛡️

Fingerprint suppression

Suppress known false positives by SHA-256 fingerprint. Stable across scans, team-shareable via config.

🐹

Single binary

Written in Go. No runtime dependencies. Cross-platform. One binary, put it on PATH and run.

Get Started

Install & run in under a minute.

From source
$ git clone https://github.com/omkoli/GQLS-CLI $ cd GQLS-CLI && go build -o gqls ./cmd/gqls $ mv gqls /usr/local/bin/gqls   # Run your first scan $ gqls scan --url https://api.yourapp.com/graphql   # With auth header $ gqls scan --url https://api.yourapp.com/graphql \           --header 'Authorization: Bearer eyJ...'   # Paste a curl from DevTools $ gqls scan --curl 'curl https://... -H "Authorization: Bearer eyJ..."'

Requires Go 1.24+. Binaries for Linux, macOS, and Windows available on the releases page ↗
